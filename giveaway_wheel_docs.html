Introducción

Giveaway Wheel es una aplicación diseñada para gestionar sorteos durante streams. Permite registrar participantes desde el chat de Twitch, seleccionar ganadores mediante una ruleta visual y asignar premios obtenidos desde la API de MegaMU.

Configuración inicial

Pasos para configurar la aplicación
  1. Configurar el OAuth de Twitch.
  2. Introducir el nombre del canal donde se realizará el sorteo.
  3. Definir el comando de participación (por ejemplo !join).
  4. Configurar la API de MegaMU introduciendo DV y API Key.
  5. Configurar los Webhooks de Discord para anunciar sorteos.

Configuración de Twitch

Cómo obtener el OAuth
  1. Ir a https://twitchtokengenerator.com.
  2. Seleccionar Chat Bot Token.
  3. Autorizar tu cuenta de Twitch.
  4. Copiar el OAuth generado.
oauth:xxxxxxxxxxxxxxxxxxxxxxxx

Configuración API MegaMU

Cómo configurar la API de MegaMU

Para conectar la aplicación con MegaMU necesitas generar una API Key desde el panel del servidor.

1️⃣ Entra en tu cuenta: https://megamu.net
2️⃣ Abre el DV Panel
3️⃣ Ve a la pestaña API
4️⃣ Pulsa Generate New Key
5️⃣ Copia la clave generada

Pegar la clave generada en el campo:

API Key (MU)

También deberás introducir el valor:

DV (es exactamente tu ID de login para entrar en la pagina de megamu)

La aplicación utiliza esta API para obtener los premios disponibles mediante:

dvapi.php?action=getawards

Discord Webhooks

Cómo crear un webhook
  1. Abrir Discord.
  2. Ir a Configuración del servidor.
  3. Entrar en Integraciones.
  4. Seleccionar Webhooks.
  5. Pulsar Crear Webhook.
  6. Copiar la URL generada.
https://discord.com/api/webhooks/XXXX/XXXX
Tipos de Webhooks usados por la app
Tipo Descripción
Live Webhook Avisa cuando el stream está en vivo.
Prize Webhook Anuncia un nuevo sorteo y los premios disponibles.

Debug Panel

Cómo abrir el panel

Dentro de la aplicación pulsa:

F12
Funciones disponibles
Función Descripción
Crear usuarios falsos Genera participantes para pruebas.
Simular comando Simula que un usuario escribió !join.
Forzar ganador Selecciona manualmente un ganador.
Reset cooldown Discord Permite reenviar webhooks.

Flujo del sorteo

  1. Un usuario escribe !join en el chat.
  2. El usuario se añade a la lista de participantes.
  3. El streamer pulsa Sortear.
  4. La ruleta selecciona un ganador.
  5. El ganador confirma en el chat.
  6. Se realiza el sorteo del premio.
  7. El resultado se guarda en el historial.
